# cn

Merges Tailwind CSS class names, resolving conflicts via `tailwind-merge` and handling conditionals via `clsx`. Use this anywhere you need to combine or conditionally apply Tailwind classes.

```tsx
import { cn } from '@capitaltg/vero';
```

## Signature

```ts
cn(...inputs: ClassValue[]): string
```

## Example

```tsx
import { cn } from '@capitaltg/vero';

// Merge static and conditional classes
cn('px-4 py-2', isActive && 'bg-primary text-white', className)

// Resolve conflicts — last value wins
cn('text-red-500', 'text-blue-500') // → 'text-blue-500'
```
