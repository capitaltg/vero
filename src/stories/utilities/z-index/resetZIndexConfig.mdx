# resetZIndexConfig

Resets all z-index values to their defaults. Useful in tests to isolate configuration between test cases.

```ts
import { resetZIndexConfig } from '@capitaltg/vero';
```

## Signature

```ts
resetZIndexConfig(): void
```

## Example

```ts
import { resetZIndexConfig } from '@capitaltg/vero';

afterEach(() => {
  resetZIndexConfig();
});
```
