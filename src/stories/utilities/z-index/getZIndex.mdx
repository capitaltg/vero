# getZIndex

Returns the z-index value for a given overlay type, applying an optional per-instance override. Used internally by overlay components — available if you need the same value in custom components.

## Parameters

  <thead>

## Example

```ts
import { getZIndex } from '@capitaltg/vero';

// Returns the configured default for 'dropdown'
const z = getZIndex('dropdown');

// Returns 999 regardless of the configured default
const z = getZIndex('dropdown', 999);
```

Per-component `zIndex` props on overlay components (e.g. `<Select zIndex={999} />`) call this internally, so you only need `getZIndex` directly when building custom overlays.
