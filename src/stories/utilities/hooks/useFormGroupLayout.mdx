# useFormGroupLayout

Returns a `className` string for form group containers. Used internally by `CheckboxGroup`, `RadioGroup`, and `SwitchGroup` — available for consumers building custom group components.

## Parameters

  <thead>
    <tr>
      <th>Name</th>
      <th>Type</th>
      <th>Default</th>

## Example

```tsx
import { useFormGroupLayout } from '@capitaltg/vero';

function MyOptionGroup({ orientation = 'vertical', columns = 1, children }) {
  const layoutClasses = useFormGroupLayout({ orientation, columns });
  return <div className={layoutClasses}>{children}</div>;
}
```
