# validateFormControlProps

Validates that a form control has either a `label` prop or an `aria-label` attribute — but not both. Throws a descriptive error at runtime if the constraint is violated. Use this in custom form controls to enforce the same accessible labeling pattern used across the library.

```tsx
import { validateFormControlProps } from '@capitaltg/vero';
```

## Signature

```ts
validateFormControlProps(componentName: string, label?: ReactNode, ariaLabel?: string): void
```

## Throws

- If neither `label` nor `aria-label` is provided
- If both `label` and `aria-label` are provided

## Example

```tsx
import { validateFormControlProps } from '@capitaltg/vero';

const MyInput = React.forwardRef(({ label, 'aria-label': ariaLabel, ...props }, ref) => {
  validateFormControlProps('MyInput', label, ariaLabel);
  // ...
});
```
