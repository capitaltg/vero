# Design Tokens

Complete reference for all design tokens in the Vero component library. These tokens power the visual design system and can be customized to match your brand.

## What are Design Tokens?

Design tokens are the foundational design decisions of the Vero component library. They define colors, spacing, typography, border radius, and animations that create a consistent visual language across all components.

All tokens are available through Tailwind CSS classes and can be customized via CSS custom properties. This allows for easy theming and customization while maintaining design consistency.

### Available Token Categories

- **Colors** - Base colors, semantic colors, and color variants for all UI states
- **Border Radius** - Consistent rounded corner values for components
- **Animations** - Predefined animations and keyframes for component interactions

## Colors

Vero uses a semantic color system with base colors and semantic variants. All colors are available as Tailwind classes (e.g., `bg-primary-400`) and CSS custom properties.
