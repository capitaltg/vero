# configureZIndex

Sets global z-index defaults for any or all overlay types. Call this once during app initialization, before any components render.

All overlay components in Vero (`Select`, `Combobox`, `Popover`, `Tooltip`, `Dialog`, `DropdownButton`) use a shared z-index system configured through these functions.

```ts
import { configureZIndex } from '@capitaltg/vero';
```

## Signature

```ts
configureZIndex(config: Partial<ZIndexConfig>): void
```

## Example

```ts

configureZIndex({
  popover: 1000,
  tooltip: 2000,
  dialog: 3000,
  dropdown: 1500,
});
```
