# getZIndexConfig

Returns a copy of the current z-index configuration.

```ts
import { getZIndexConfig } from '@capitaltg/vero';
```

## Signature

```ts
getZIndexConfig(): ZIndexConfig
```

## Example

```ts
import { getZIndexConfig } from '@capitaltg/vero';

const config = getZIndexConfig();
// → { popover: 1000, tooltip: 2000, overlay: 50, dialog: 3000, dropdown: 1500 }
```
